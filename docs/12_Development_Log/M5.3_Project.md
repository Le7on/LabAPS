# Development Log

## Milestone M5.3 — Project

**Project:** Laboratory Advanced Planning & Scheduling Platform (Lab APS)

**Version:** 1.0

**Status:** Complete

**Owner:** Development Team

**Completed:** 2026-07-08

---

# 1. Objective

Add the Project entity to the Laboratory Definition domain and make Demand
reference a real Project, so demand is tied to a validated business object.

---

# 2. Scope

- `Project` entity (project code, name, active) + ORM + repository.
- Use cases: create project, list projects; wired into the Unit of Work.
- API: `POST/GET /api/v1/projects`.
- `AddDemandUseCase` now validates that the referenced project exists (404
  otherwise).
- Alembic migration for the project table.
