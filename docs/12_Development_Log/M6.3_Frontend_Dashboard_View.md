**Project:** Laboratory Advanced Planning & Scheduling Platform (Lab APS)

**Version:** 1.0

**Status:** Complete

---

# 1. Objective

Add a Dashboard view that consumes the reporting endpoint (M8.1) and becomes the
default landing page, closing the reporting loop end to end.

---

# 2. Scope

- `src/api/reports.js`: `getDashboard()`.
- `src/views/DashboardView.vue`: summary cards for the dashboard counts.
- Route `/dashboard` as the default landing; nav link in the app shell.

---

# 3. Deliverables

| Component      | Location                                  |
| -------------- | ----------------------------------------- |
| Reports API    | `frontend/src/api/reports.js`             |
| Dashboard view | `frontend/src/views/DashboardView.vue`    |
| Router / shell | `frontend/src/router/index.js`, `App.vue` |

---

# 4. Out of Scope

- Charts / trends; drill-down into entities.

---

# 5. Acceptance Criteria

- The dashboard loads counts from `GET /api/v1/reports/dashboard`.
- It is the default landing route.
- The frontend builds and passes ESLint + Prettier.
