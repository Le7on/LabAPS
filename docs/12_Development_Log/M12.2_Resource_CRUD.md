---

# 1. Objective

Allow created data to be edited and deleted, not just added and deactivated.

---

# 2. Scope

- Update + delete for Project, Equipment, Staff; delete for Workflow — across
  repository, use case and API (PUT + DELETE). Update replaces fields and
  many-to-many relations.
- Deleting a Project that still has workflows returns 409 (guarded).
- Frontend: Edit (pre-filled slide-over) and Delete (confirm) actions on all
  laboratory list pages.
- Seeded the six standard machines SU-HM-01/02/09/10/12/13 (idempotent script).
